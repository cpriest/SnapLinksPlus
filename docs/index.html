About Snap Links

Check out our tutorial page for a quick introduction to using SnapLinks v3

Snap Links is a Firefox addon that lets you draw a lasso around links, checkboxes and other elements and do things with them.

  • Open links in New Tabs
  • Check/Uncheck checkboxes
  • Select radio items

Other actions in the not too distant future planned (from v2)

  • Open Tabs in a New Window
  • Bookmark All
  • Download All
  • Copy Links to Clipboard
